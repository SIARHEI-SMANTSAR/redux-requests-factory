# Legacy example using `redux-requests-factory` with `create-react-app` and `redux-observable`

This example is kept for older Create React App and redux-observable setups. It
uses React 16, Create React App 3, redux-observable 1, RxJS 6, and TypeScript
3.7.

For the current simple Create React App example, use
[`simple-modern`](../simple-modern).

## Download manually

Download the example:

```bash
curl https://codeload.github.com/SIARHEI-SMANTSAR/redux-requests-factory/tar.gz/master | tar -xz --strip=2 redux-requests-factory-develop/examples/create-react-app/with-redux-observable-legacy
cd create-react-app/with-redux-observable-legacy
```
